Dunny Fonts

Welcome to Dunny Fonts, a collection of typefaces designed for the D’ni script. D’ni is a constructed language from the Myst video game universe, originally published by Cyan in 1993.

View on GitHub →

Dunny Pen NTS

Dunny Pen NTS is a D’ni script font. It takes inspiration from the classic look and feel of D’ni broad-nibbed pen scripts, and provides an alternative to Cyan's offical Dnifont or to D'ni Script by Colin Arenz and Sebastian Ochs. This font currently uses NTS mapping for letters to D'ni sounds.
